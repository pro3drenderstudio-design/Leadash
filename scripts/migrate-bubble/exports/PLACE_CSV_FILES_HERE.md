# Bubble CSV Exports

Place your Bubble.io CSV exports here with these exact filenames:

| Bubble data type    | Filename                  |
|---------------------|---------------------------|
| User                | `User.csv`                |
| LeadCampaign        | `LeadCampaign.csv`        |
| LeadRecord          | `LeadRecord.csv`          |
| CreditTransactions  | `CreditTransactions.csv`  |
| SupportMessage      | `SupportMessage.csv`      |

## How to export from Bubble

1. Open your Bubble app editor
2. Go to **App Data** (left sidebar)
3. Select a data type (e.g. "User")
4. Click **Download as CSV**
5. Repeat for each type above
6. Rename/place the files here

Files listed in this folder are ignored by git (see .gitignore).
